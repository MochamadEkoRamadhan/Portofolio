## 🔧 Customization Tips

### Mengubah Warna Utama
Di `style.css`, cari dan ganti:
```css
#667eea  → warna baru untuk indigo
#764ba2  → warna baru untuk purple
```

### Mengubah Font
Di atas CSS, ganti:
```css
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
```

### Menambah Project Baru
1. Salin struktur `.project-card`
2. Update gambar dan judul
3. Salin modal template `modal5`
4. Update ID modal dan deskripsi
