Automatic Water Filling Control

This automatic water gallon filling dashboard is designed to make it easy to control and monitor the filling process. From the control menu, users can select 5-liter or 15-liter fill capacity, stop the filling with a stop button, and calibrate sensors to keep measurements accurate. All functions are presented in a modern, clean glassmorphism interface. The dashboard also provides real-time monitoring information. Data such as current water volume, cost estimates, daily usage totals, and revenue are shown clearly, with interactive line charts to visualize trends. Additional features like a data reset button help manage records, making the system more efficient and transparent.

  • Automatic filling control with 5L and 15L capacity
  • Stop button to halt filling
  • Sensor calibration for accurate measurements
  • Modern glassmorphism-style dashboard
  • Real-time water volume monitoring
  • Interactive line chart for trend visualization
  • Automatic cost and revenue estimation
  • Data reset button for record management

AC Motor Speed Control

This ESP32 WebServer system for controlling a meatball maker motor lets users adjust motor speed precisely using a TRIAC-based dimmer slider with zero-crossing detection. With IR sensor integration to read motor RPM and an E18 sensor to count meatballs, the system can monitor production in real time through a modern, responsive web interface. Through the web dashboard, users can view RPM charts, produced meatball counts, and automatic revenue estimates. Additional features such as PID parameter settings (Kp, Ki, Kd), configuration storage, and a counter reset make the system flexible and easy to use. With an interactive Chart.js interface, production monitoring becomes more practical, transparent, and efficient.

  • Motor speed control with TRIAC dimmer slider
  • Zero-crossing detection for high precision
  • Real-time RPM reading with IR sensor
  • Automatic meatball counting
  • Modern responsive web dashboard
  • Interactive RPM and production charts
  • PID parameter settings (Kp, Ki, Kd)
  • Automatic configuration storage

Hardware

ESP32, 220V AC motor, TRIAC, IR sensor, E18 photoelectric sensor

Backend

Arduino IDE, embedded C++, PID algorithm, PWM control

Frontend

HTML5, CSS3, JavaScript, Chart.js for production charts

Automatic Production Counter

This program is an ESP32-based IoT production counter system designed to count items passing a sensor, detect rejected items, and automatically display the total good items. Count data is stored safely in internal memory (Preferences), so it persists even after restarts. The system is also integrated with a Telegram bot, enabling users to monitor total items, rejects, and the ESP32 IP address directly via chat. In addition, it includes an interactive web dashboard that shows gauge charts and real-time trend charts using Chart.js. Users can add manual counts, reset the counter, and send data to Google Spreadsheet for further records and analysis. With a mix of local control and cloud integration, this system provides modern, transparent, and efficient production monitoring.

  • Automatic item counting with sensors
  • Real-time rejected item detection
  • Secure data storage in EEPROM/Preferences
  • Telegram bot integration for notifications
  • Web dashboard with gauge charts
  • Interactive item count trend charts
  • Sync to Google Spreadsheet
  • Manual count add and counter reset

Hardware

ESP32, E18 photoelectric sensor, push button, status LED, relay

Backend

Arduino IDE, C++, Telegram API, Google Sheets API

Frontend

HTML5, CSS3, JavaScript, Chart.js for data visualization

Integration

Telegram Bot API, Google Sheets API, HTTP requests

IoT Oyster Mushroom Monitoring

This ESP32-based IoT system is designed to monitor environmental conditions for oyster mushroom cultivation in real time. Using temperature, humidity, and CO2 sensors, the system collects data continuously and displays it on a responsive web dashboard. When environmental parameters exceed or fall below optimal thresholds, the system sends alert notifications to farmers, enabling quick adjustments. Historical data is stored for trend analysis, helping growers optimize their cultivation environment for maximum yield. This solution brings smart agriculture technology to mushroom farming, improving both efficiency and crop quality.

  • Real-time temperature and humidity monitoring
  • CO2 level tracking for optimal growth
  • Alert notifications when thresholds are breached
  • Interactive web dashboard with live gauges
  • Historical data logging and trend charts
  • ESP32 IoT integration with WiFi
  • Automated misting control based on humidity
  • Low-power mode for continuous operation

Hardware

ESP32, DHT22 temperature/humidity sensor, CO2 sensor, relay module

Backend

Arduino IDE, embedded C++, HTTP REST protocol

Frontend

HTML5, CSS3, JavaScript, Chart.js for trend visualization

Integration

ESP32 WebServer, local network access, alert notifications
